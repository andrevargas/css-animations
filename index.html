Transição de propriedades

As transições do CSS fornecem uma forma de controlar a velocidade das animações ao alterar propriedades CSS de um elemento.

Ao invés das mudanças de propriedades serem efetuadas de imediato, você pode fazer com que as mudanças aconteçam ao longo de um período de tempo.

element { transition-duration: 1s; transition-property: background-color; transition-timing-function: linear; }

Animações com
a regra @keyframes

As animações CSS tornam possível animar transições de um estilo para outro. As animações consistem em dois componentes: um estilo descrevendo a animação e um set de keyframes que indicam quais propriedades o elemento possuirá em determinados estados.

element { animation-name: fade-in; animation-timing-function: linear; animation-duration: 1s; } @keyframes fade-in { from { opacity: 0; } to { opacity: 1; } }
Este é provavelmente o balão mais inútil de toda a web...

A Regra de Ouro!

Para garantir que as animações sejam sempre fluídas, e beirem aos 60 quadros por segundo (FPS), há uma única regra a ser seguida:

Anime apenas as propriedades opacity e transform.

As consequências em alterar propriedades além dessas serão de sua responsabilidade.
